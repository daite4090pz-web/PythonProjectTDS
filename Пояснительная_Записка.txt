1. Описание проекта
Tower Defence Simulator (TDS) — это стратегическая игра в жанре Tower Defence, разработанная на Python с использованием библиотеки Arcade. Игроку предстоит защищать свою базу от волн врагов, строя и улучшая различные виды башен на карте. Проект включает расширенную систему геймплея с несколькими уровнями сложности, тремя уникальными картами, 15 волнами врагов и тремя типами боссов.

2. Системные требования
ОС: Windows 7/8/10/11 (64-bit)
Процессор: 1.6 GHz или выше
Оперативная память: 512 MB RAM
Место на диске: 60 MB свободного места
Дополнительно: Звуковая карта, клавиатура, мышь


3. Установка и запуск
Запуск из готового исполняемого файла:
Скачайте архив с игрой
Распакуйте архив в любую папку на компьютере
Запустите файл TDS.exe из распакованной папки
Запуск из исходного кода (для разработчиков):
Установите Python 3.14 или выше с сайта python.org

Установите библиотеку Arcade:
pip install arcade

Обязательно!
